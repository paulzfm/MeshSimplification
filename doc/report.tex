\section{综述}

实现基于边坍塌 (edge-collapse) 的网格简化 (mesh simplification) 方法。

\section{功能实现}

我们实现了以下功能：

\begin{itemize}
    \item 保持拓扑结构的简化；
    \item 支持两种收缩点的选择：(1) 中点；(2) 线性优化。
\end{itemize}
